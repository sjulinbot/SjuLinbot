# 🎓Linbot - SJU Portal

## 📌 Overview

SJU Portal is a web application developed for St. Joseph's University to manage academic activities like attendance, results, and fees in one centralized platform.

It also includes **SjuLinbot**, a rule-based chatbot that helps users navigate services easily using predefined options.

---

## 🚀 Features

* 📊 Attendance management system
* 📅 Academic calendar
* 💰 Fees tracking
* 🤖 Rule-based chatbot (SjuLinbot)
* 📢 Admission updates ticker

---

## 🛠 Tech Stack

* Frontend: React (Vite)
* Backend: Node.js
* Data Storage: JSON (data.json)

---

## ⚙️ Setup Instructions

```bash
git clone https://github.com/sjulinbot/SjuLinbot.git
cd SjuLinbot
npm install
npm run dev
```

---

## 📁 Project Structure

* `src/` → React components
* `server/` → Backend logic
* `public/` → Static assets
